User Guide

Overview

Hi, I'm Dan, a coffee lover. We often think of brewing coffee as using an automatic drip machine where the process is hands-off for the most part. However, coffee enthusiasts have been exploring other brewing methods, such as pourover, French press, and AeroPress, to achieve the best cup possible.

This application is my personal, non-profit, open-source project that aims to help new coffee enthusiasts experiment with some of my favourite coffee brewing recipes. Links to the original source are provided in each recipe. Espresso-based drinks are not yet covered.

I hope this application helps you find your perfect cup!

Features

Library of Methods and Associated Recipes

Brew methods and their associated recipes are uploaded manually to the database. The long-term goal is to allow users to add or improve recipes based on their testing. Here is a list of supported recipes organized by brew method:

Hario V60 Pourover Method:
  • James Hoffmann V60 1 cup
  • Pilot Coffee Roasters Pourover Brew Guide
  • Subtext Coffee Roasters V60 Brew Guide
French Press Method:
  • Pilot Coffee Roasters French Press Brew Guide
AeroPress Method:
  • Pilot Coffee Roasters AeroPress Brew Guide
  • Subtext Coffee Roasters AeroPress Brew Guide

Recipe Information

The recipe name and description is displayed at the top along with a reference link (if applicable). The following key sections are then provided.

Equipment

List of equipment recommended to carry out each recipe.

Brew Parameters

Key parameters for the setup:

  • Total Coffee
  • Total Water
  • Brew Ratio (Water Weight / Coffee Weight)
  • Grind Size (of coffee beans)
  • Temperature (of water)
  • Total time (of brew)

Preparation

List of steps to ensure the water, ground coffee beans and equipment are ready before beginning the timed brewing process.

Brew

List of timed steps to brew the coffee. The target total water and time at the end of each step is listed. The total water is the cumulative amount of water added since the start of the brew steps. The total time is the target elapsed time at the end of each step.

Settings

Enable Read Steps Aloud: When enabled, the current step to complete is read aloud. For brew steps, the target total water and time are read aloud after the step description.

Enable Voice Commands: When enabled, the user can use the application by saying commands rather than pressing buttons or using the keyboard. The word detected by the application will appear in the Last Detected Command display. See Main Controls section for more details on supported commands.

Main Controls

The main controls are located at the bottom of the application along with the Elapsed Time during the Brew steps. The commands change according to the current stage of the recipe. A command can be executed by pressing a button, pressing a key or by saying a keyword if Enable Voice Command is checked.
